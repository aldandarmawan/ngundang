## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/aldandarmawan/ngundang.git
cd ngundang
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run locally
```bash
npm run dev
```

The site will be available at `http://localhost:4321` by default.
